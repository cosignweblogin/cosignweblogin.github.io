Jarod

I figured it out - I don't know if something is wrong with the install
package or what - but the problem was the registry - the install puts

ConfigPath as c:\program files\IIScosign\cosign.dll.config

however - when I ran that DBmon.exe - it looked like it was looking for
ConfigFile (not ConfigPath).  So I manually added the registry key of

ConfigFile as c:\program fiels\IIScosign\cosign.dll.config

and it worked like magic.
