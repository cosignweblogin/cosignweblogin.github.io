The Cosign Development Team is pleased to announce Cosign 0.9.2 . As you saw in our previous announcement, cosign is now in production on campus, for Conferencing on the Web, aka COW. https://weblogin.umich.edu/ also has a new look, so go check it out!

The 0.9.2 release is mostly bug fixes and UI tweaks/overhauls. One important change is we now have a central page to serve as the "CosignPostErrorRedirect" -- https://weblogin.umich.edu/post_error.html

Additionally, since logout is now global ( for all cosign protected services ) we now ask for confirmation ( with explanatory text ) on logout.

As always, please direct any questions to cosign@xxxxxxxxx

Thanks,
The Cosign Dev Team
