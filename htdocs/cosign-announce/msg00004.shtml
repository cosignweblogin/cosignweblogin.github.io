IISCosign 1.0.0 Release Candidate 2 is now available from <http://weblogin.org>. (RC1 was released hastily this morning and lacked the MSXML 4.0 installer and an updated notes.txt file.) Besides many important bug fixes there is also an installer! This is being declared "RC2" so we can have feedback on the installer, in particular, and the filter, as always, before making an official release.

If you'd like to be part of the IISCosign discussion in general you can go to <http://directory.umich.edu> and subscribe to the iiscosign.codesign e-mail list.

Thanks to everyone who's helped us get this far!

--Jarod Malestein
