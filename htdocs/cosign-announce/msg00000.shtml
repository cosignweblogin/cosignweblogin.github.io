The IIS Cosign Filter is now up to version 0.8.1.  This adds a logging
capability.  Here are the instructions for creating a log file:

If you would like to keep logs of the Cosign Filter's activity, add the
following registry entry (the trailing backslash is very important!):

    LogFilePath:
    C:\Program Files\CosignFilter\Logs\


This will provide more information for those having trouble getting IIS
Cosign to run.  You can download 0.8.1 here:

http://www.umich.edu/~umweb/software/cosign/
