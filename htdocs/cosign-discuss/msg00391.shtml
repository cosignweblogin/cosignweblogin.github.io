Hey All,

The Cosign Dev team has been playing around with mod_cosign in an effort to make things like "Allow from" and "Satisfy" and all those other native Apache AuthZ type things very cosign compatible. We think we've got a pretty good stab at it for Apache 1.3, and would like to solicit either actually conf files or theoretical situations based on actual usages to test against to make sure, in the meantime, we haven't *lost* any preexisting CosignProtected On/Off functionality. We'll fine tune the integration using these examples as our testing regime, and then ask that the example-bearers, if they have a test environment in which they can try out the CVS version of the module, do so as well.

I would really like to make this 1.8.1, or maybe 1.8.5, but in order to feel good adding something potentially wacky to a "point" release i want to have it thoroughly vetted, and I don't know the ways you all may be be using any particular directive, Make sense?

No need to reply to the whole list ( unless you want to ), just send config files/snippets/ideas to cosign@xxxxxxxxxx The Apache 2 version for this should follow shortly, and we won't make whatever release we're gonna make until both filters do this right. :) I just want to get a jump on testing what we do have working.

Thanks,
Johanna and the Cosign Dev Team
