The Cosign Team would like to announce the release of Cosign 0.9.1 ( MD5(cosign-0.9.1.tar.gz)= cd9682cf45f02ed7faa88ef740480638 ).

This is mainly a bug fix release, though there are a few small additions. The cgi, for instance, now confirms that there is no KDC spoofing a foot ( thanks to Dan Hyde for help with this ).
