On 18 Jan 2005, at 15:07, Jim Zajkowski wrote:
We want to put up a site that is instantly available from a UM IP address, or requires Cosign from outside those addresses.

"Satisfy" might work:

	http://httpd.apache.org/docs/mod/core.html#satisfy
	http://httpd.apache.org/docs-2.0/mod/core.html#satisfy

Like this:

	Require valid-user
	Allow from 192.168.1
	Satisfy Any

You might also need to specify CosignAllowPublicAccess, and you might need a URL that requires authentication only. Please let us know if this works for you.
