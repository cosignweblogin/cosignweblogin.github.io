Yes, this is possible.  You'll need to do two things.  First, apply the
IISCosign at the "web sites" level in IIS Manager (rather than to each site
individually).  Second, add a new <Service /> tag for each web site that will
be Cosign-protected.  Restart IIS for the new settings to take effect.
