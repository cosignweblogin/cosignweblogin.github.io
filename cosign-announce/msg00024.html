IISCosign 1.2.0 beta available

  • To: cosign-announce@xxxxxxxxx
  • Subject: IISCosign 1.2.0 beta available
  • From: Jarod Malestein <jarod@xxxxxxxxx>
  • Date: Tue, 01 Mar 2005 15:59:30 -0500
  • Sender: jarod@xxxxxxxxxxxxxx

For those interested in the Kerberos 5 ticket-retrieving feature, it is now available for testing. Many thanks to Mark Earnest at Penn State for writing this part of IISCosign!

<http://www.umich.edu/~umweb/software/cosign/download.html>

This is a dll only. You will probably want to run the 1.1.1 installer before using 1.2.0 beta.

Add at least the following line to your cosign.dll.config file:

<GetKerberosTickets>FALSE</GetKerberosTickets>

If you want this set to TRUE you'll need:

<GetKerberosTickets>TRUE</GetKerberosTickets>
<KerberosTicketsPath>c:\Kerb5tkts</KerberosTickets>

With the TRUE setting you'll have a new server variable, HTTP_KRB5CCNAME. The value is set to FILE:[KerberosTicketsPath]\[TicketFile]. Please report any problems, concerns, or successes to cosign@xxxxxxxxxx
