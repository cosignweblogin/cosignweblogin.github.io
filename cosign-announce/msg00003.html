The Cosign Development Team is pleased to announce Cosign 1.0.0. The UM weblogin server ( weblogin.umich.edu ) has been upgraded to 1.0.0 in production.

The 1.0.0 release contains a few tweaks and changes that were not present in the 0.9.4 release. Most significant among these are:

o the splash screen ( warning user's redirected from a service to visit weblogin first ) has been removed.
o HTTP cache and browser expiration headers have been improved across multiple Windows, MacOS, and X Windows browsers.
o Case #2 ( user visits service first ) has been simplified to REGISTER the user before returning to the referring service the first time.
o improved autoconf support for cgi

While not strictly part of the cosign software, the weblogin.org site has been completely overhauled for easier navigation.

As always, please direct any questions to cosign@xxxxxxxxx

Thanks,
